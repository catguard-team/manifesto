## [v1.4.0] — 2026-04-30

세계로 나아가기 천명 — 김구·도산 안창호 정신 명시.

### Changed

- `MANIFESTO.md` "할 수 있는 것" 🌐 항목: "영어/다국어 번역" → "**영어/다국어 번역 + 세계로 나아가기**". 김구 선생·도산 안창호 선생의 정신을 자경단의 세계 활동 근거로 명시 — 임시정부가 한국 밖에서 한국을 지켰듯, 흥사단이 청년 한 명부터 인격을 세웠듯, **동네에 뿌리내린 채** 세계로 손을 뻗는다. v1.2.0의 "정보 교환 정도" 수동 표현을 능동으로 전환.
- `.github/RECRUITMENT.md` 3곳 (트윗②, 본문 "우리가 안 하는 일", 디스코드 모집문): "영어 번역으로 글로벌 진출"을 "글로벌 브랜딩으로 동네 떠나기"로 교체. 매니페스토의 실제 레드라인과 일치시킴 — 도구가 아닌 의도를 금지.
- `manifesto/CLAUDE.md`, `manifesto/.github/copilot-instructions.md` "영어/다국어 번역" 절: "만들어도 됩니다" → "**적극 만듭니다**". "글로벌 진출용 마케팅 톤 금지" → "**글로벌 브랜딩으로 동네 떠나기** 톤 금지" + 구체 예시("Korea's leading…", "World-class…").
- `.github/profile/README.md` FAQ: Q1(후원) 톤 타이트하게 정리. 새 Q "한국 밖에서도 활동하나요?" 추가 — 김구·도산 정신과 레드라인("글로벌 브랜딩으로 동네 떠나기 ❌") 한 답변에. Q "코드 못 짜는데 도울 수 있나요?" 항목에 "번역" 추가 (번역 적극 추진과 일치).

### Notes

- 레드라인 그대로: ❌ "글로벌 브랜딩으로 동네를 떠나기" — 김구·도산 두 분도 정체성을 떠난 적 없음. 세계로 나간 건 한국을 위해서였다는 점을 매니페스토에 명시.
- "거대한 약속(전 세계, 모든 어린이…)" 금지 조항(MANIFESTO §무엇을 안 하는가)은 유지 — 세계 활동 ≠ 거대 약속.

### Changed

- `MANIFESTO.md`: ❌ "영어 번역으로 글로벌 진출" → ❌ "글로벌 브랜딩으로 동네를 떠나기"로 좁힘. 도구가 아니라 의도를 금지.

- 에이전트 지침(`CLAUDE.md`, `.github/copilot-instructions.md`): "영어 번역본 금지/후원 안 받음" 잘못된 단언 제거.

자경단 출범. 0번 자경단원이자 단장 묘냥이(@GoGoComputer)가 첫 영역 선포.

### Added

- `MANIFESTO.md` v1.0 — 자경단의 정체성·목적·한계 선언.
- `GANGNYANG.md` v1.0 — 행동강냥 10조 본문 + 해설. 부칙(위반·이탈·강령 외) 포함.
- `SAFETY.md` v1.0 — 새끼고양이 안전 정책. 개인정보 절대 금지 목록, 미성년자/장애인/노묘 추가 보호, 만들지 않는 자동화 목록, 사고 대응 절차.
- `OPERATIONS.md` v1.0 — 의사결정/합류·이탈/작업 흐름/채널 분리/후원·재정/외부 협력/분쟁 해결/자경단 종료 규칙.

### Decided

- 자경단의 1차 영역: 한국어 디지털 골목.
- 자경단의 단위: "옆집 한 명" 정도.
- 자경단의 슬로건: "기술, 이제 모두의 골목이다" / "귀찮지만 우리가 한다" / 사인오프 "냐-옹."
- 자경단의 끝: 자경단이 필요 없을 때.

### Notes

- 첫 강령이라 어설프다. 강령 8조에 따라 부끄러워하지 않는다.
- 개정 PR 환영. 단, 시민단체 톤은 솜방망이.

## 개정 기록 양식

향후 개정은 다음 형식으로 추가한다.

```markdown
## [v1.x] — YYYY-MM-DD

한 줄 요약 (왜 바꿨는지).

### Added / Changed / Deprecated / Removed / Fixed / Security

- 변경 내역 한 줄 (영향 범위 표기)
```

### 카테고리 선택 가이드

- **Added**: 새 조항·문서·정책.
- **Changed**: 기존 조항·정책의 표현·범위 조정.
- **Deprecated**: 폐지 예정. 즉시 제거 아님.
- **Removed**: 폐지 완료.
- **Fixed**: 오타·링크·맞춤법.
- **Security**: `SAFETY.md` 강화. **완화는 없다**.
