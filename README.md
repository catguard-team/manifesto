# manifesto

> 자경단의 헌법, 매니페스토, 안전 정책. 이 레포가 자경단의 머리뼈다.

기술 골목의 평화와 자유는 우리가 지킨다. 라고 써놓긴 했지만 거창하게 받아들이지 마라. 이 레포는 동네 길고양이 무리가 "우리는 이 정도 선까지 한다"를 적어둔 메모 묶음이다. 강령 개정 PR 환영. 단, 시민단체 톤으로 오면 솜방망이로 돌려보낸다.

## 문서

| 파일 | 무엇 |
|------|------|
| [`MANIFESTO.md`](./MANIFESTO.md) | 자경단이 누구이고 왜 있는지 |
| [`GANGNYANG.md`](./GANGNYANG.md) | 행동강(綱)냥(領) 10조 본문 + 해설 |
| [`SAFETY.md`](./SAFETY.md) | 새끼고양이 안전 정책 (강화는 즉시, 완화는 안 됨) |
| [`OPERATIONS.md`](./OPERATIONS.md) | 자경단 운영 규칙 |
| [`CHANGELOG.md`](./CHANGELOG.md) | 강령 개정 이력 |

## 읽는 순서

처음 합류한 사람은 위에서부터 읽어라. 시간 없으면 [`MANIFESTO.md`](./MANIFESTO.md) 한 개만 읽고 디스코드 합류해도 된다. 나머지는 업무 중 필요할 때 들춰보면 된다.

## 개정 절차

- **오타·링크·맞춤법**: 그냥 PR.
- **표현 다듬기**: PR + 이유 한 줄.
- **강령 본문 변경**: 디스코드 토론 → 자경단원 과반 의견 수렴 → PR. 단독 결정 금지.
- **SAFETY.md 완화**: 절대 안 됨. 강화는 단장 단독 결정 가능.

전부 [`CHANGELOG.md`](./CHANGELOG.md)에 한 줄로 남긴다.

## 어울리지 않으면

읽다 "이 톤 별로네"라고 느꼈으면 자경단이 안 맞는 사람이다. 그만두는 자유는 강령 어딘가에 박혀 있다. 굳이 찾지 말고, 그냥 가도 된다.
