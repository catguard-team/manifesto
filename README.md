# manifesto

> 자경단의 행동 약속, 매니페스토, 안전 정책이 모인 레포입니다. 자경단의 정체성 문서가 여기에 있어요.

"기술 골목의 평화와 자유는 우리가 지킨다"라고 써 두었는데요, 거창하게 받아들이지 않으셨으면 좋겠습니다. 이 레포는 동네 길고양이 모임이 "우리는 이 정도 선까지 한다"를 적어둔 메모 묶음 정도라고 봐주시면 돼요. 강령 개정 PR 환영합니다. 다만 시민단체 톤은 자경단 결과 잘 맞지 않아서 정중하게 다른 표현으로 부탁드릴 수 있습니다.

## 읽는 순서

처음 합류하신 분은 위에서부터 읽으시면 됩니다. 시간이 부족하시면 [`MANIFESTO.md`](./MANIFESTO.md) 하나만 읽고 디스코드에 합류하셔도 됩니다. 나머지는 작업 중 필요하실 때 들춰보시면 돼요.

## 개정 절차

- **오타·링크·맞춤법**: 그냥 PR 보내주시면 됩니다.
- **표현 다듬기**: PR + 이유 한 줄.
- **강령 본문 변경**: 디스코드 토론 → 자경단원 과반 의견 수렴 → PR. 단독 결정은 하지 않습니다.
- **SAFETY.md 완화**: 하지 않습니다. 강화는 단장 단독 결정 가능합니다.

전부 [`CHANGELOG.md`](./CHANGELOG.md)에 한 줄로 남깁니다.

## 결이 안 맞으시면

읽다가 "이 결 안 맞는데" 싶으시면, 자경단이 잘 맞지 않는 모임일 수 있어요. 그만두실 자유는 강령 어딘가에 박혀 있습니다. 굳이 찾지 않으셔도 되고, 그냥 가셔도 됩니다.
