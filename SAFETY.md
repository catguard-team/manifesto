# SAFETY — 새끼고양이 안전 정책

> 자경단의 모든 활동에서 이 문서는 다른 모든 문서보다 우선합니다.
> 강화는 즉시 적용합니다. 완화는 하지 않습니다.

---

## 0. 원칙

새끼고양이의 안전은 자경단의 자랑·효율·홍보 그 어떤 것보다 위에 있습니다. 안전과 다른 가치가 충돌하면 안전을 택합니다. 그래서 자경단의 진도가 느려져도 괜찮습니다. 자경단은 빠른 것이 자랑이 아니라고 봅니다.

---

## 1. 절대 금지 — 개인정보

다음은 자경단의 어떤 코드, 문서, 커밋 메시지, 이슈, 디스코드 메시지에도 **들어가지 않습니다**.

| 종류 | 예시 | 처리 |
|------|------|------|
| 실명 | "○○○ 어린이가..." | 가명/이니셜로 |
| 학교명 | "△△초등학교 3학년" | "초등학교 저학년" 정도까지만 |
| 주소 | "서울시 ○○구 ○○동" | 시 단위 또는 "수도권/비수도권"까지만 |
| 연락처 | 전화번호, 카톡 ID, 이메일 | 절대 금지 |
| 생년월일 | 정확한 날짜 | 연령대(예: "70대")까지만 |
| 사진 | 얼굴 식별 가능한 사진 | 본인+보호자 동의 + 모자이크 권장 |
| 음성 | 이름·주소가 들리는 녹음 | 절대 공개 금지 |
| 기기 정보 | 시리얼, MAC, IMEI | 코드 어디에도 박지 않음 |

이미 들어가 있는 게 발견되면 **즉시 제거 + git history 정리**(BFG Repo-Cleaner). 단순 force-push로는 부족하다.

→ 단계별 절차: [`LEAK_CLEANUP.md`](./LEAK_CLEANUP.md). 토큰 폐기 → 백업 → BFG → force-push --mirror → GitHub 캐시·fork·미러 정리 → 회고.

## 2. 미성년자 추가 보호

### 2-1. 보호자 동의 원칙

- 미성년자(만 18세 미만)와의 모든 비대면 1:1 활동은 **보호자 사전 동의**가 있어야 한다.
- 동의 기록은 자경단 어디에도 보관하지 않는다(개인정보). 멘토 자경단원 본인이 별도 보관하고, 자경단에는 "동의 받음" 사실만 남긴다.

### 2-2. 1:1 금지 채널

- 디스코드 DM, 카카오톡 1:1, 전화 — **자경단 미성년자와 사용 금지**.
- 모든 소통은 보호자가 같이 있는 자리, 또는 공개 채널(디스코드 공개방, 부모 카톡방).

### 2-3. 작품 공개 시

- [`jwidori`](https://github.com/catguard-team/jwidori) 등재는 본인 + 보호자 동의 둘 다 필요.
- 가명 사용이 기본. 본인이 실명 공개를 강력히 원해도 미성년자는 가명.
- 등재 후 본인/보호자가 삭제 요청 시 **즉시 삭제**(git history 포함). 이유 묻지 않음.

## 3. 장애인·노묘(어르신) 추가 보호

### 3-1. 동의 능력

- 인지 기능 저하가 있는 어르신, 의사결정 보조가 필요한 장애인의 경우 **법적 대리인/가족 동의** 추가 확보.
- 동의 받기 전에는 콘텐츠 등재·인터뷰·사진 촬영 금지.

### 3-2. 접근성 보장

- 자경단의 자료가 새끼고양이에게 도달 가능한가? — 글씨 크기, 색 대비, 스크린리더 호환, 한국어 평이성 검토.
- "쉬워요!"라는 표현 금지. 실제로 어려운 사람이 자책한다.

## 4. 절대 만들지 않는 것

자경단은 다음 종류의 자동화·도구를 **만들지 않는다**.

- 송금/결제 자동화 (자경단원 명의로 새끼고양이 대신 결제 등)
- 비밀번호 저장/대입 도구
- 본인인증 우회 도구
- 새끼고양이 명의 계정 자동 생성/관리
- 위치 추적 (자경단원 간에도 금지)
- 통화/메시지 무단 녹음·기록

"새끼고양이 편의를 위해서"라는 명분이 아무리 강해도 만들지 않는다. 편리함은 통제권 상실의 입구다.

## 7. 사고 발생 시

새끼고양이 안전 관련 사고(개인정보 노출, 보호자 동의 누락 등)가 발생/의심되면:

1. **즉시 멈춰라.** 추가 작업 금지.
2. **단장에게 알려라.** 디스코드 DM 또는 GitHub 이슈(private).
3. **확산 차단.** 노출된 자료를 모든 채널에서 제거(git history 포함). → 절차: [`LEAK_CLEANUP.md`](./LEAK_CLEANUP.md)
4. **새끼고양이/보호자에게 알려라.** 사실대로. 변명·축소 금지.
5. **회고를 남겨라.** 단, 회고 자체에 개인정보가 안 들어가게.

자경단은 사고를 숨기지 않습니다. 단, 새끼고양이의 사생활은 사고보다 우선합니다. 회고는 식별 정보를 모두 제거한 형태로만 공유합니다.
