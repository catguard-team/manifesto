# manifesto/CLAUDE.md

> 자경단 정체성 문서 레포. AGENTS.md를 먼저 읽고 이 파일을 보세요.

## 이 레포의 역할

자경단의 헌법, 매니페스토, 안전 정책 등 **정체성 문서**를 보관합니다. 코드는 없습니다. 모든 변경은 자경단의 사상에 영향을 주므로 신중히.

## 작업 시 주의

1. **톤이 가장 중요**: AGENTS.md의 "톤 가이드" 절대 준수
2. **변경 이력 보존**: 큰 변경은 CHANGELOG.md에 기록
3. **자경단원 합의**: 강령 본문 변경은 단독 결정 금지, 디스코드 토론 후 PR

## 파일별 편집 권한

| 파일 | 누가 수정? |
|------|-----------|
| `MANIFESTO.md` | 단장 + 자경단원 합의 |
| `GANGNYANG.md` | 단장 + 자경단원 과반수 |
| `SAFETY.md` | 단장 단독 (강화는 즉시, 완화는 절대 안 됨) |
| `OPERATIONS.md` | 단장 + 자경단원 |
| `CHANGELOG.md` | 모든 PR이 자동 추가 |

## 금지 사항

- 강령 조항 삭제 금지 (개정만 가능)
- "사명감", "헌신", "위대한" 같은 시민단체 단어 사용 금지
- 영어 번역본 만들지 마세요 (요청 있을 때만 별도 처리)

## 자주 쓰는 명령

```bash
# 변경 검토
git diff GANGNYANG.md

# 변경 이력 보기
git log --oneline GANGNYANG.md

# 외부인 시각으로 다시 읽기
cat MANIFESTO.md | less
```
